# Jarvis AI Assistant

A Python-based AI virtual assistant inspired by JARVIS.

## Features
- Voice Recognition
- Text-to-Speech
- Open Websites
- Launch Applications
- Search Google & YouTube
- Weather Updates
- AI-powered Responses
- Custom Commands

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- OpenAI API
- Tkinter
- Requests

## Installation

pip install -r requirements.txt

## Run

python main.py
